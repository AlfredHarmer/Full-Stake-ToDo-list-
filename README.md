## Full-Stack To-do List

## Description

A Full-Stack application built with React, Node.js, Express, and SQLite.
The project demonstrates core full-stack concepts including REST APIs, state management and persistent data storage.

Users can add and view task stored in a SQL database, these task can then be edited by the user, checked as complete when a task has been
forfilled or removed from the database completely.

## Features

- View a list of Taks fetched from a backend API
- Add new Taks via a React frontend
- Edit exsiting task from the database that then remain with the  
  persistant data
- Task can be fully removed from the SQL database
- Persistent data storage using SQLite
- RESTful API built with Express
- Cross-origin communication using CORS

## Technologies Used

- React
- Node.js
- Express
- SQLite
- CORS
